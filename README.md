# nane

## Для запуска проекта нужен `yarn` или `npm`
```
yarn install
npm install
```

### Компиляция и перезагрузка при изменении кода (режим разработки)
```
yarn serve
npm run serve
```

### Подготовка в продакшен
```
yarn build
npm run build
```

### Lints файлы
```
yarn lint
npm run lint
```

### Деплой на github pages (Внимание! Поменяйти данные на свои в ././deploy и ./vie.config)
```
yarn deploy
npm run deploy
```
